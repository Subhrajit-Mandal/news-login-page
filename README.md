# British Times – Login Page Design

## Overview
This project presents a modern and minimal login page design for a news platform called "British Times".

## Design Focus
- Clean and professional layout
- Strong typography inspired by traditional news media
- User-friendly login interface
- Balanced use of colour and spacing

## Features
- Minimal UI for clarity and ease of use
- Visually appealing gradient background
- Clearly structured input fields and call-to-action

## Tools Used
- Figma (UI/UX Design)
- html (skeleton development)
