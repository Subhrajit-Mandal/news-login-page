# British Times – Login Page and Home page

## Overview
This project presents a modern and minimal login page and homepage for a news platform called "British Times".

## Design Focus
- Clean and professional layout
- Strong typography inspired by traditional news media
- User-friendly interface

## Features
- Minimal UI for clarity and ease of use
- Clearly structured input fields and call-to-action

## Tools Used
- Figma (UI/UX Design)
- html (skeleton development)
